Create an append-only activity log that records all project management events with timestamps and actor information. Events to track include: story creation/updates, task status changes (created, in-progress, done, blocked), epic lifecycle events (created, marked done, archived), assignment changes, point estimates, and any other state mutations. The log should provide a chronological view of "what happened and when" across the project, enabling audit trails and team visibility into project activity.
